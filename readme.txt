Данный, довольно базовый, инструмент, создан с целью упростить создание шаблона для картотек в ролевый игр!

Так же этот инструмент упрощает работу со стилизацией текста под кодовые блоки в качестве показа значений, или же при помощи двоиточек!

Так же, инструмент поддерживает редактирование и добавление собственных шаблонов для картотек (редактировать файл "forms.js" в папке "forms" по инструкции заданной там же)!
